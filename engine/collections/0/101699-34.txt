In 2016 , Grease premiered on Royal Caribbean 's Harmony of the Seas and Independence of the Seas . The Harmony of the Seas original cast included Joe Giacone ( Danny ) , Laura Sillett ( Sandy ) , Ross James McKnight ( Vince Fontaine / Teen Angel ) , Nick Hayes ( Kenickie ) , Sarah Drake ( Rizzo ) , Oliver Jacobson ( Roger ) , Emma Salvo ( Jan ) , Curtis Brown ( Sonny ) , Rebecca Stenhouse ( Marty ) , Mike Eborall ( Doody ) , Leigh Jacobson ( Frenchy ) . The 2017 cast achieved Royal Caribbean 's highest guest ratings . Notable cast members included Kory Fulton ( Danny ) , Wesley Carpenter ( Sandy ) , Kassandra Haddock ( Rizzo ) , Drew Arisco ( Roger ) , Dominic Fortuna ( Teen Angel / Vince Fontaine ) . The Independence of the Seas select original cast members included Cory Lloyd ( Danny ) , Katrina Diehm ( Sandy ) , Dominic Fortuna ( Vince Fontaine / Teen Angel ) , Maxim Gukhman ( Kenickie ) , Stephanie Miller ( Rizzo ) , Kristian Morse ( Roger ) , Louise Willoughby ( Jan ) .
