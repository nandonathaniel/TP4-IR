Registry permissions Permission Description Query Value The right to read the registry key value . Set Value The right to write a new value Create Subkey The right to create subkeys . Enumerate Subkeys Allow the enumeration of subkeys . Notify The right to request change notifications for registry keys or subkeys . Create Link Reserved by the operating system . Delete The right to delete a key . Write DACL The right to modify permissions of the container 's DACL . Write Owner The right to modify the container 's owner . Read Control The right to read the DACL .
